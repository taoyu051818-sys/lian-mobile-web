## Safe usage

Use handoffs to understand what a previous thread attempted, which files it touched, what validation it claimed, and what risks it left behind.

Do not use handoffs alone to decide what is currently implemented, whether a task is accepted today, what repo owns a file today, or what command should be run today.

Acceptance must be recorded by reviewer validation in a current task board/task doc/PR or newer PR-derived status file.
